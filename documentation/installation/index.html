Getting Started

Step 1 of 3

Installation & Setup

Get started with Roblox Studio and create your first project.

Download Roblox Studio

Roblox Studio is the official IDE for creating Roblox games. It's completely free and available for Windows and macOS. Head over to the Roblox website and click the "Create" button to download Roblox Studio.

💡 Tip: Make sure your computer meets the minimum requirements: Windows 7 or later, macOS 10.7 or later, 512MB RAM minimum (1GB+ recommended).

Create Your First Game

Open Roblox Studio and you'll see the start screen. Click on a template to create a new game, or start with a blank project. For beginners, we recommend starting with the "Baseplate" template.

Understanding the Interface

The Roblox Studio interface has several key areas:

Viewport (Center)

This is where you see your game world in 3D. You can navigate using the mouse and keyboard.

Explorer (Left)

Shows the hierarchy of all objects in your game. You can expand and collapse folders to organize your workspace.

Properties (Right)

Shows all properties of the selected object. You can modify these to change how objects behave.

Output (Bottom)

Shows debug messages and errors from your scripts. This is essential for debugging.

Insert Your First Part

Let's create a simple object. Go to Insert > Part and place it in the game world. You'll see it appear in the Explorer on the left. Try changing its properties like Size, Color, and Position.

Create Your First Script

Right-click on a Part and select "Insert Object > Script". A new script will be created inside that part. Double-click it to open the editor.

-- This is your first script!
local part = script.Parent
part.BrickColor = BrickColor.new("Bright red")
print("Hello, Roblox!")

Test Your Game

Click the "Play" button in the top toolbar to test your game. You'll see a new window open with your game running. Click "Stop" when you're done testing.

💡 Tip: Use Ctrl+P to quickly toggle between play and edit mode.
