Roblox API Fundamentals

Understand the core Roblox API for game development.

The Game Hierarchy

Roblox games have a hierarchical structure. The "game" object is the root, containing DataModel with services and game content.

Services

Services are essential systems in Roblox. Access them using game:GetService(). Common services include:

Players Service

Manages players in your game.

local Players = game:GetService("Players")

local function onPlayerAdded(player)
    print(player.Name .. " joined!")
end

Players.PlayerAdded:Connect(onPlayerAdded)

RunService

Connects to game loops (Heartbeat, RenderStepped).

local RunService = game:GetService("RunService")

RunService.Heartbeat:Connect(function(deltaTime)
    -- Code here runs every frame
    print("Frame!")
end)

Workspace

Contains all physical objects in the game.

local Workspace = workspace

local part = Instance.new("Part")
part.Parent = Workspace

UserInputService

Detects player input (keyboard, mouse, etc).

local UserInputService = game:GetService("UserInputService")

UserInputService.InputBegan:Connect(function(input, gameProcessed)
    if gameProcessed then return end
    if input.KeyCode == Enum.KeyCode.Space then
        print("Space pressed!")
    end
end)

Creating Objects

Create new instances using Instance.new(). Objects need a parent to exist in the game.

local part = Instance.new("Part")
part.Shape = Enum.PartType.Ball
part.Size = Vector3.new(1, 1, 1)
part.Position = Vector3.new(0, 5, 0)
part.BrickColor = BrickColor.new("Bright red")
part.Parent = workspace  -- Make it visible

Properties

Objects have properties that define their behavior. Common part properties:

local part = Instance.new("Part")
part.Size = Vector3.new(2, 1, 3)
part.Position = Vector3.new(0, 10, 0)
part.CanCollide = true
part.TopSurface = Enum.SurfaceType.Smooth
part.BottomSurface = Enum.SurfaceType.Brick
part.Parent = workspace

Events

Events fire when something happens. Connect to events to respond to changes.

Touched Event

Fires when a part touches another part.

local part = script.Parent

part.Touched:Connect(function(otherPart)
    print(otherPart.Name .. " touched me!")
end)

Changed Event

Fires when a property changes.

local part = script.Parent

part:GetPropertyChangedSignal("Size"):Connect(function()
    print("Size changed to " .. tostring(part.Size))
end)

Vectors and CFrames

Vector3 represents 3D positions. CFrame (Coordinate Frame) represents position and rotation.

local position = Vector3.new(0, 5, 0)
local cframe = CFrame.new(Vector3.new(0, 5, 0)) * CFrame.Angles(0, math.rad(45), 0)

local part = Instance.new("Part")
part.Position = position
part.CFrame = cframe
part.Parent = workspace

Best Practices

Always parent objects last so they render immediately. Cache service references. Use local variables for frequently accessed objects. Check if instances exist before using them.
